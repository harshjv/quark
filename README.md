# Quark Logger (for electron)

> WIP

## Installation

npm install --save quark-logger


## Usage

    const Quark = require('quark-logger')
    const quark = Quark('example:main')

    quark('log something')

    // ..
    // ...
    // ..

    Quark.showWindow.show() // to show logs in a window
